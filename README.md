*Global Online Democracy*

**Install**

in root run ```npm i```

in ```/client``` run  ```npm i```

In the root directory add ```.env``` file.

use the following configuration:
```
GOOGLE_CLIENT_ID=<>
GOOGLE_CLIENT_SECRET=<>

PASSPORT_SECRET=<>

JWT_SECRET=<>

CLOUDINARY_PAAS=<>

```

**Run**

Run local mongodb local server

in root run ```npm run tsc```

in root run ```npm run tsc```

in client run ```npm start```
